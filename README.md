# iGaming Analytics

A modern analytics dashboard for iGaming operators.

## Features

- Real-time analytics
- Player behavior analysis
- Revenue tracking
- Platform health monitoring
- Custom reporting
- AI-powered insights

## Getting Started

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
4. Build for production:
   ```bash
   npm run build
   ```
